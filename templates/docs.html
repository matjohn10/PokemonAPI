Introduction

Welcome to the Pokémon API documentation. This API provides basic information about Pokémon species, including their names, types, and strengths and weaknesses.

Responses

The API responds with JSON objects. To vizualise the objects returned, please visit the home page and test the API's capabilities.

Base URL

The base URL for all API requests:

http://127.0.0.1:5000/api/

Endpoints

The following endpoints are available:

- Get All Informations

Get a list of all information available in this API.

GET /

- Get All Pokemons

Get a list of all pokemons

GET /pokemons

- Get Pokémon by ID

Get detailed information about a specific Pokémon by its ID.

GET /pokemons/{id}

- Get Pokémon by Name

Get detailed information about a specific Pokémon by its name.

GET /pokemons/{name}

- Get All Descriptions of Pokemons

Get a list of detailed descriptions of all pokemons

GET /description

- Get Specific Description of Pokemon

Get the detailed description of a pokemon

GET /description/{name or id}

- Get All Pokemons per Type

Get a list of types objects that contain every pokemon of that type.

GET /types

- Get Pokemons for a Specific Type

Get a list of all pokemons of that type.

GET /types/{type}

- Get Types Combat Strengths and Weaknesses

Get a list of types object that contains the types they are strong against and the types they are weak againts.

GET /types-combat

- Get Specific Type Combat Strengths and Weaknesses

Get the type's types they are strong against and the types they are weak againts.

GET /types-combat/{type}
